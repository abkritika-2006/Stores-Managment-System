# Stores-Managment-
**Stores Management System** is a Java-based inventory and sales management application built using OOP principles. It enables user authentication, product management, stock tracking, and transaction recording. The project follows a layered architecture (Model, DAO, Service, Utility) for better scalability, maintainability, and organization.

## Features

- Secure user authentication and login system
- Admin and cashier role management
- Add, update, delete, and search products
- Inventory and stock quantity tracking
- Sales transaction processing
- Transaction history management
- Password hashing for enhanced security
- File-based data storage and persistence

## How to Run

1. Clone the repository.
2. Open the project in VS Code or any Java IDE.
3. Compile the source files.
4. Run `App.java`.
5. Log in using the default admin account or create a new cashier account.
